To add a new user(s)
~~~~~~~~~~~~~

<-

Click the tab "Benutzer_verwalten"
Click "Benutzer hinzufügen"
Enter the Username
Enter the Userpassword
Click "yes" for adminrights

->

RESTART THE SERVER (click "VBMessenger 9 Server Beenden)

--> Now a client is able to connect with the username+password!
